# GH-200: Automate Your Workflow with GitHub Actions (End-to-End Demo)

This repository demonstrates an end-to-end CI/CD flow:
- Node.js app with tests
- CI builds + uploads ZIP artifact
- Docker image built and pushed to **GHCR** and **Azure Container Registry (ACR)**
- Deployment to **Azure Web App for Containers** using the ACR image
